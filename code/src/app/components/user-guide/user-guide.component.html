User Guide

This page will provide a step-by-step guide on how to use ChartGenious.

  1. Select a file with a dataset (.xlsx, .xls, or .csv).
  2. View the raw data in a paginated table.
  3. Select a chart type and apply filters:
    • Sort data
    • Trim rows
    • Change colors
    • Add text below the chart
    Repeat for more charts if needed.
  4. Edit layout:
    • Display 2 charts in a column or 3 in a row
    • Edit chart titles
    • Change background color
  5. Convert the layout to PDF and download it.
